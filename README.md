# Internship Application Management API

A RESTful API built using Node.js and Express.js for managing internship applications.

## Features

- Create Application
- View All Applications
- View Single Application
- Update Application Status
- Delete Application
- Search Applications By Role
- Email Validation
- Duplicate Email Check

## Technologies Used

- Node.js
- Express.js
- CORS

## API Endpoints

GET /applications

GET /applications/:id

POST /applications

PUT /applications/:id

DELETE /applications/:id

GET /applications?role=Backend Developer
